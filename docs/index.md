## Цель практики

Сформировать практические навыки соадминистрирования и автоматизации баз данных путём самостоятельной разработки информационной системы в выбранной предметной области.

## Задачи

1. Спроектировать реляционную базу данных (5–10 таблиц) с правильными связями и нормализацией.
2. Создать представления (VIEW) для аналитических выборок.
3. Написать хранимые процедуры для автоматизации типовых операций.
4. Реализовать триггеры для контроля целостности и аудита.
5. Настроить пользовательские роли СУБД с разграничением прав.
6. Реализовать хэширование паролей пользователей.
7. Продемонстрировать запросы с JOIN, LEFT JOIN, RIGHT JOIN.
8. Реализовать ИС с графическим интерфейсом и CRUD-операциями.
9. Подготовить отчёт с не менее чем 10 скриншотами.
